Personal Projects

Favourite Movies

Full-stack web app using Flask, SQLAlchemy, and bootstrap. Users can register for an account and log in. It uses the OMDb api to allow users to search and favourite movies.

F2 Racing

Browser game where the player controls a car and must dodge oncoming cars, made with Javascript, JQuery, HTML/CSS. I implemented a collision detection algorithm to trigger the game over function when the user hits a car.

Student Productivity App

Full-stack web app using React, Express, Node.js, PostgreSQL. Users can add and delete daily tasks to the interface. They can use the timer, built using React hooks, to take study breaks or time their study sessions.

Discord Bot

Discord bot using python and discord.py library. The bot sends meeting information if the user uses a certain command. There is also a profanity filter implemented using a list of banned words and a for loop.
